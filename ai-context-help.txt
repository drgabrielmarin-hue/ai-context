Produce AI context-file for GitHub project, directory, or YouTube video.

Usage:
  ai-context [flags]
  ai-context [command]

Available Commands:
  stats       Show file statistics including estimated LLM token count.

Flags:
      --debug             Enable debug logging
  -e, --exclude strings   Exclude files matching globs (e.g., 'tests,docs')
  -f, --file string       File with list of URLs to process
      --for-ai            AI-friendly output (plain text, piped input)
  -h, --help              help for ai-context
  -i, --include strings   Include files matching globs (e.g., '*.go,*.md')
  -s, --max-size int      Maximum file size in bytes to include (default 10MB) (default 10485760)
  -t, --threads int       Number of threads to use for processing (default 10)
  -v, --version           version for ai-context

Use "ai-context [command] --help" for more information about a command.
